Software

Even if you have the software installed already please read the instructions and make sure you have the right versions. The rest of this document is split into two sections installing and updating.

  • Node.js v0.10.x
  • Mongodb v2.4.x
  • Robomongo v0.8.4
  • Postman
  • Chrome browser
  • node-inspector v0.7.4
  • nodemon v1.2.1
  • modulus v3.0.2

Installing

Installing Node.js

If you do not have node.js on your laptop
  1. Go to http://nodejs.org/ and click the install button on the webpage
  2. Open the downloaded file and follow the instructions in the installer
  3. Once installed open up the command line and type $ node –v it should return v0.10.33

Installing Mongodb

If you do not have mongodb on your laptop

  1. Go to http://www.mongodb.org/ and click the “Download MongoDB 2.6” button then select the download for your OS
  2. Open the downloaded file and follow the instructions in the installer
  3. Once installed open up the command line and type $ mongod –version it should return db version 2.6.5

Installing Robomongo

Download the application from http://www.robomongo.org/ and follow the install instructions. The current version is 0.8.4

Installing Chrome

If Chrome is not already installed on your laptop, download the browser from http://www.google.co.uk/intl/en_uk/chrome/browser/index.html. Chrome is needed for debugging node and running the Postman app.

Installing Postman

Postman is a “Packaged Chrome App” and as such runs within Chrome. If you do not have Postman on your laptop:

  1. In the Chrome browser go to http://www.getpostman.com/ and click the download button, then follow the install instructions in the Chrome Web Store.

Installing node modules

If you do not have node-inspector or nodemon your laptop, once your node.js is installed, open a command line and type

  1. $ npm install -g node-inspector
  2. $ npm install -g nodemon

You may have to install these modules using the sudo command i.e. $ sudo npm install -g nodemon

Installing modulus

At the end of the workshop I will be showing you how to deploy an API to the web. I will do this on the https://modulus.io/ hosting platform. If you wish to follow along and deploy your work to web you will need to setup an account. Modulus offers $15 free credit for you to try the service, which should last a month.

  1. Create an account at https://modulus.io/register
  2. Type $ npm install -g modulus into a command line to install the deploy client. You may need to use sudo command i.e. $ sudo npm install -g modulus

Updating

Below are the instructions for updating software and tools if you have older versions installed on your laptop.

Updating Node.js

First check the version number. Any version from 0.10.x greater will be OK.

  1. Open up the command line and type $ node –v it should return a version number i.e. v0.10.33
  2. If you have a lower version then 0.10.x please update node.js using the method you first installed it with. This could be with Homebrew or another package management system or it could have been installed directly from http://nodejs.org/
  3. Once updated please check the version again using the command $ node –v

Updating Mongodb

Any version from 2.4.x greater will be OK.

  1. To check the version type $ mongo –version into the command line.
  2. If you have a lower version please update Mongodb using the method you first installed it with. This could be with Homebrew or another package management system or it could have been installed directly from http://www.mongodb.org/
    Please backup any data you have in Mongodb before upgrading
  3. Once updated please check the version again using the command $ mongo –version

Updating Robomongo

Robomongo is a standalone application the current version is 0.8.4. If you need to upgrade just download and install the newer version from http://www.robomongo.org/

Updating node modules

To make sure you have the current version of the node modules run the following commands:

  1. $ npm update -g node-inspector
  2. $ npm update -g nodemon
  3. $ npm update -g modulus

Updating Chrome and Postman

You should not need to upgrade Chrome or Postman as they are upgraded automatically.
